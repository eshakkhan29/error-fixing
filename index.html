How to javascript work?

JavaScript is a high level dynamic programming language. It works on the browser. It works synchronously. And it works line after line. It can only do one thing at a time. And its callstack is a single callstack.

difference between local storage and session storage

Localstorage : Local storage is what is stored in the browser. It lasts during the season and lasts when the session ends. If you open the browser or cut the tab and restart it, data is available. Its capacity is 5 MB to 10 MB.

Sessionstorage : Session storage is what works during the session. When the season is over, it is over. It can no longer be accessed later. Its storage is 5 MB.

what is Event loop

JavaScript is a single threatened programming language. Everything that is written in JavaScript runs one after the other. But if it takes a long time to execute a function, then it executes the next one. Not all work is done by JavaScript alone. Some work is done by browser V8 engine. This is the event loop that the browser is serially making one call after another. And it work (Lifo) last in first out.
